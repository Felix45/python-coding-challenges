## Mergesort coding challenge
> Given an unsorted array of integers, return a sorted array in non decreasing order.
You must write an algorithm with `O(nlog n)` runtime complexity.

## Example I
```Python
Input: nums = [6, 3, 5, 1]
Output: [1, 3, 5, 6]
```
## Example II
```Python
Input: nums = [-1,-3,0,2]
Output: [-3, -1, 0, 2]
```
## Example III
```Python
Input: nums = [-1,4,-5,-6]
Output: [-6, -5, -1, 4]
```
